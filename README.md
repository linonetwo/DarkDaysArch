# DarkDayArch

An Architecture helper and mod content Designer for Cataclysm-DarkDaysAhead.

## Development

`npm run` following scripts, for example, `npm run tauri:dev`

```shell
ui:start
ui:build
ui:test
tauri:dev
tauri:build
tauri:help
```
